## Junghwan (Steve) Kim — Blog

Personal Jekyll blog with notes on tech, finance, and other explorations.

### View the live site

https://baobabtr33.github.io/blog

### Run locally

Requires Ruby and Bundler.

```bash
bundle install
bundle exec jekyll serve
```

Then open http://localhost:4000/blog/ in your browser.

### Structure

- `_posts/` — published posts
- `_layouts/` / `_includes/` — Jekyll templates
- `tech/`, `finance/`, `miscel/` — category landing pages
- `aboutme.md` — about page
- `draft/` — local drafts (gitignored)
